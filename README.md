# QT_opencv

## 简介
这个项目结合了 QT 和 OpenCV，旨在提供一个图形界面应用程序，用于处理和显示图像。

## 功能
-  V1.0 :图像加载和显示
- V1.1 :
    1. 增加进入系统界面切换主系统窗口
    2. 以及加入无功能的界面元素
- V1.2 :
    1. 增加色块识别的功能

## 依赖

- OpenCV 4.10
